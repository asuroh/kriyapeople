# Kriya People

## Release Note

### Run the server

- Pull from the branch
- Make sure .env is correct
- Run below commands

```bash
go mod vendor
cd server
go run main.go
```

### Database Setup

- Run below query :
```
    run db in file file\db.sql
```

### Postman : 
Postman collection : 
    in file Kriya People.postman_collection.json

Postman Env : 
    in file Local.postman_environment.json
